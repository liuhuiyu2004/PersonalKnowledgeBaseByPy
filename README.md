# 个人知识库系统

一个功能完整的本地知识库管理系统，支持 Web 界面和 API 交互，可自动从网络采集信息并使用 AI 整理。

## ✨ 特性

- 📚 **知识管理** - 创建、编辑、删除知识条目
- 🏷️ **标签系统** - 灵活的标签分类和管理
- 🔍 **全文搜索** - 快速检索知识库内容
- 🌐 **网络采集** - 抓取网页内容并自动保存
- 🔎 **网络搜索** - 在网络上搜索相关信息
- 🤖 **AI 处理** - 自动生成摘要、提取标签
- 📊 **统计分析** - 查看知识库使用情况
- 🎨 **美观界面** - 现代化的 Web UI

## 🚀 快速开始

### 1. 安装依赖

```bash
install.bat
```

### 2. 配置

```bash
copy .env.example .env
```

编辑 `.env` 文件配置必要参数（如 OpenAI API Key）。

### 3. 启动服务

```bash
start.bat
```

### 4. 访问

- **Web 界面**: http://127.0.0.1:8000
- **API 文档**: http://127.0.0.1:8000/docs

## 📖 主要功能

### Web 界面

1. **知识列表** - 查看所有知识条目，支持分页
2. **新建知识** - 手动添加知识，支持富文本编辑
3. **搜索** - 关键词搜索，支持标签和来源筛选
4. **网络采集** - 
   - 网页抓取：输入 URL 自动抓取内容
   - 网络搜索：搜索相关信息并保存（支持 DuckDuckGo 和 Bing）
5. **标签管理** - 自定义标签名称和颜色
6. **统计** - 查看知识库统计数据

### 核心功能

- ✅ 富文本编辑器（支持格式化、图片、链接等）
- ✅ 全文搜索（标题、内容、摘要）
- ✅ 标签分类系统
- ✅ 网页自动抓取
- ✅ 多引擎网络搜索
- ✅ AI 自动生成摘要
- ✅ 知识图谱可视化

## 🔧 技术栈

- **后端**: FastAPI + SQLAlchemy + SQLite
- **前端**: Vue 3 + Element Plus
- **爬虫**: httpx + BeautifulSoup4
- **AI**: LangChain + OpenAI API (可选)
- **图表**: ECharts

## 📝 使用说明

### 创建知识

1. 点击"新建知识"
2. 输入标题和内容（支持富文本）
3. 添加标签
4. 可选：使用 AI 生成摘要
5. 点击"保存"

### 网络采集

**网页抓取：**
1. 输入网页 URL
2. 勾选"自动保存"和"AI 生成摘要"
3. 点击"开始抓取"

**网络搜索：**
1. 输入搜索关键词
2. 选择搜索引擎（DuckDuckGo 或 Bing）
3. 设置结果数量
4. 勾选"自动保存"
5. 点击"开始搜索"

### 搜索知识

1. 点击"搜索"标签
2. 输入关键词
3. 可选：按标签或来源筛选
4. 查看搜索结果

## ⚙️ 配置说明

### 必需配置

- `database_url`: 数据库连接（默认 SQLite）

### 可选配置

- `openai_api_key`: OpenAI API 密钥（启用 AI 功能）
- `host`, `port`: 服务监听地址

## ❓ 常见问题

**Q: AI 功能无法使用？**  
A: 检查是否正确配置 `openai_api_key`。

**Q: 网络爬虫抓取失败？**  
A: 某些网站有反爬机制，建议更换目标网站。

**Q: 如何备份知识库？**  
A: 备份 `knowledge_base.db` 文件即可。
